BiblioSense | The Tactile Media Library
Welcome to the landing page repository for BiblioSense, an upcoming, beautifully constrained physical home for your digital cultural life, designed meticulously for macOS.
This repository contains the static HTML landing page showcasing the application's vision, features, and unique UI/UX philosophy.
About BiblioSense
In an era of endless streaming queues and fragmented libraries, BiblioSense acts as a personal tracking system. 
It aggregates all your media types into one interface, allowing you to track everything you watch, read, play or listen to locally.

Supported Media:
- Movies & TV Shows
- Books & Audiobooks
- Video Games
- Music & Podcasts

Core Features Highlighted

- The Unified Space: Allows you to track watchlists, log reading progress, manage game backlogs, and queue podcast series all in one place.
- Notes and Rating: A journaling system with a markdown editor, a  half-star ratings system, to record your impressions about your media.
- A Keyboard-First Navigation system : By pressing ⌘K, a command palette descends, allowing you to search TMDB, IGDB,Google Books and more without your hands ever leaving the keyboard.
- Smart Collections & Up Next: Organize media the way you want with drag-and-drop and a smart dashboard that shows exactly what to consume next.
  
 Design Philosophy

The landing page—just like the upcoming app—is built on a hybrid design system combining Neumorphism and Glassmorphism:

- Soft Extrusions (Neumorphism): Elements have physical presence.
- Frosted Canvases (Glassmorphism): Overlays, navigation, and journaling panels utilize background blurs.

The landing page is designed to be lightweight, fast, and completely static:
- HTML5: Pure, semantic markup.
- Tailwind CSS (via CDN): For rapid, utility-first styling and custom UI shadows/blurs.
- Lucide Icons: For SVG iconography.
